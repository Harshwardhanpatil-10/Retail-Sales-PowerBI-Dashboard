# Retail Sales Performance Dashboard (Power BI)

This project analyzes retail sales transaction data using Power BI to generate business insights and answer key analytical questions.

## Dashboard Objectives

1. Identify Top and Bottom 5 products by Sales, Profit, and Quantity Sold.
2. Analyze Sales Trends over time (Daily, Monthly, Quarterly, Yearly).
3. Understand the relationship between Sales and Profit.
4. Compare Sales/Profit/Quantity between two user-selected time periods.
5. Analyze Average Discount across promotion categories.
6. Track Total Number of Orders.
7. Provide detailed order-level transaction analysis with filters.
8. Visualize Sales distribution across different cities.

## Key Features

- Interactive filters and slicers
- Multi-page dashboard design
- Period comparison analysis
- Geographic sales visualization
- Product performance analysis
- Transaction-level data exploration

## Tools Used

- Power BI
- DAX
- Data Modeling
- Data Visualization

## Dashboard Pages

### Overview Dashboard
Shows high-level business insights including sales by city, profit vs sales relationship, and overall sales trends.

### Product Performance Analysis
Highlights top and bottom performing products based on sales, profit, and quantity sold.

### Period Comparison Dashboard
Allows comparison of sales, profit, and quantity between two selected time periods.

### Order-Level Analysis
Displays detailed transaction data with filters for product, date, customer, and promotion category.

## Key Insights

- Apple iPhone 14 generated the highest revenue and profit.
- Weekend Flash Sale promotions resulted in the highest discounts.
- Sales performance peaked in 2023 before declining in 2024.
- Product-level analysis helps identify high-performing and low-performing products.
